# Student Grade Tracker

## Project Description

Student Grade Tracker is a Java console-based application developed as part of the CodeAlpha Java Programming Internship.

The application allows users to enter student names and grades, store them using ArrayList, and generate a summary report.

## Features

* Add student names and grades
* Store data using ArrayList
* Calculate average score
* Find highest score
* Find lowest score
* Display complete student report

## Technologies Used

* Java
* ArrayList
* Scanner Class

## How to Run

1. Compile the program:
   javac StudentGradeTracker.java

2. Run the program:
   java StudentGradeTracker
