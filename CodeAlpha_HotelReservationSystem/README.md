# Hotel Reservation System

## Project Description

Hotel Reservation System is a Java console-based application developed as part of the CodeAlpha Java Programming Internship.

This system allows users to search available rooms, make reservations, cancel bookings, view booking details, and simulate payment processing.

## Features

* View available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* View booking details
* Payment simulation
* Uses Object-Oriented Programming (OOP)
* Console-based interface

## Technologies Used

* Java
* HashMap
* Scanner Class
* Object-Oriented Programming (OOP)

## Room Categories

* Standard Room
* Deluxe Room
* Suite Room

## How to Run

1. Compile the program:

   javac HotelReservationSystem.java

2. Run the program:

   java HotelReservationSystem

## Sample Operations

* View Rooms
* Book Room
* Cancel Reservation
* View Booking Details
