# Artificial Intelligence Chatbot

## Project Description

Artificial Intelligence Chatbot is a Java console-based chatbot developed as part of the CodeAlpha Java Programming Internship.

The chatbot interacts with users and responds to frequently asked questions using rule-based Natural Language Processing (NLP) techniques.

## Features

* Interactive chatbot conversation
* Rule-based responses
* Handles greetings and common questions
* Answers questions about Java, AI, and College
* Exit command using "bye"
* Console-based interface
* Uses Object-Oriented Programming (OOP)

## Technologies Used

* Java
* Scanner Class
* Rule-Based NLP
* Object-Oriented Programming (OOP)

## How to Run

1. Compile the program:

   javac AIChatbot.java

2. Run the program:

   java AIChatbot

## Sample Questions

* Hello
* What is Java?
* What is AI?
* How are you?
* Help
* Bye
