# Stock Trading Platform

## Project Description

Stock Trading Platform is a Java console-based application developed as part of the CodeAlpha Java Programming Internship.

This project simulates a basic stock trading environment where users can view market data, buy stocks, sell stocks, and track their portfolio performance.

## Features

* View available stocks and market prices
* Buy stocks
* Sell stocks
* Track portfolio holdings
* Calculate total portfolio value
* Uses Object-Oriented Programming (OOP)
* Console-based user interface

## Technologies Used

* Java
* HashMap
* Scanner Class
* Object-Oriented Programming (OOP)

## Available Stocks

* TCS
* Infosys
* Reliance
* HDFC

## How to Run

1. Compile the program:

   javac StockTradingPlatform.java

2. Run the program:

   java StockTradingPlatform

## Sample Operations

* View Market Data
* Buy Stocks
* Sell Stocks
* View Portfolio Performance
